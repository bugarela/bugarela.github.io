    2. Chaleira

  2. Defina uma estrutura de Kripke para um sistema de caixa da água com as seguintes características:

    • A caixa da água contém uma bóia que impede que ela transborde, ou seja, a bóia interrompe o enchimento quando a caixa está completamente cheia.

    1. Fórmula: \(X(b)\)
      • [ ] \([b], [b], [a]\)
      • [ ] \([a], [b], [a]\)
      • [ ] \([a], [a], [b]\)
    2. Fórmula: \(a -> b\)
      • [ ] \([a], [a], [a]\)
      • [ ] \([a], [b], [a]\)
      • [ ] \([a, b], [a], [a]\)
      • [ ] \([b], [a], [a]\)
    3. Fórmula: \(F(a)\)
      • [ ] \([a], [a], [a]\)
